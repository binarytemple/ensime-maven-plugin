ensime:generate

Full name:

st.happy_camper.maven.plugins:ensime-maven-plugin:0.0.1-SNAPSHOT:generate

Description:

Generates ENSIME configuration files.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
  • Requires dependency resolution of artifacts in scope: test.

Optional Parameters

Name Type Since Description
formatterPreferences File - The formatter preferences
Default value is: ${basedir}/src/ensime/formatter.properties.
User property is: ensime.formatter.preferences.
skip boolean - Skip the operation when true.
Default value is: false.
User property is: ensime.skip.
